# LLM News Research Bot

## Overview
LLM News Research Bot is an AI-powered tool that allows researchers, journalists, and analysts to extract insights from multiple news articles. By inputting URLs of articles, users can ask questions, and the bot retrieves precise, source-referenced answers, ensuring transparency and credibility.

## Problem Statement
Manually analyzing multiple news sources is time-consuming and prone to misinformation. LLM News Research Bot automates this process, delivering accurate, source-backed answers instantly, enhancing research efficiency and credibility.

## Features
- **AI-Powered Query Resolution** – Get direct answers to your questions from multiple news articles.
- **Automated Information Retrieval** – Eliminates manual effort in summarization and fact-checking.
- **Source-Referenced Answers** – Ensures transparency by providing article sources for every response.
- **Optimized Research Workflow** – Enhances productivity for journalists, analysts, and researchers.

## Technology Stack
- **LLM (Large Language Model)** – For advanced natural language processing.
- **LangChain** – To manage LLM interactions and optimize responses.
- **LLAMA-3.2 by Groq** – The core AI model powering the bot.
- **Streamlit** – For building an interactive and user-friendly interface.

## Installation
```bash
# Clone the repository
git clone https://github.com/your-repo/llm-news-research-bot.git
cd llm-news-research-bot

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
```

## Usage
1. Open the application in your browser.
2. Enter the URLs of news articles.
3. Ask relevant questions about the articles.
4. Get instant, source-backed answers from the bot.
